# Heritage-Map India
An interactive data visualization tool exploring India's heritage sites using Python, Streamlit, and Pandas.

## Features
- Interactive geospatial mapping of heritage sites.
- State-wise data filtering.

## Technologies Used
- **Python** (Pandas for data processing)
- **Streamlit** (Web framework)
- **Geospatial Analysis**

## How to run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run src/app.py`
